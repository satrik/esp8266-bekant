# esp8266-bekant
Complete replacement of the IKEA BEKANT controller with an esp8266.

# I've already a working setup/code. I will push the "beta" in the next days. 

## Circuit
![wemos fritzing](/img/wemos-bekant.png)

## Parts

- Wemos D1 Mini
- MCP2003
- 3x 2.2kΩ resistor
- 1x 1.0kΩ resistor
